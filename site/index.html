Guardrails

Deterministic hooks enforce rules without relying on LLM memory

Deterministic Hook
Branch Protection
Blocks git commit, push, and merge on protected branches (main, master, staging)
PreToolUse:Bash → branch-guard.js
Deterministic Hook
Session Guard
Detects active worktrees at session start, injects context warning into conversation
SessionStart → session-guard.js
Mandatory Gate
Review Gate
Blocks PR creation unless .devflow/review-verdict.json exists and has no critical/high findings
PreToolUse:Bash → review-gate.js
Context Injection
Worktree Warning
When worktrees are active, injects reminder to work in .worktrees/ directory, not on main branch
SessionStart → stdout message
Skill Chain Gate
Verdict Bridge
Review writes verdict JSON → ship reads it. Bridges non-deterministic review with deterministic hook gate
/df:review → .devflow/ → /df:ship
Health Check
/df:doctor
Verifies all hooks installed, dimensions discovered, verdict gates working. 11 checks, zero LLM.
doctor.js → JSON report
